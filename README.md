## RNAseq workshop exercises

This repo contains exercises and presentations that are part of a two-day RNA-seq data analysis workshop presented at the Young Investigators Symposium of the Skintegrity Network. They provide a hands-on introduction to processing, exploring, and analyzing RNA-seq count data using R and Bioconductor, as well as exercises for single cell RNAseq analysis using Seurat
